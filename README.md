# 🏁 F1 Performance Analytics (SQL Project)

This project analyzes Formula 1 race results using SQL to extract insights about driver performance across multiple seasons (2008–2026).

## 📊 Project Goals
- Identify most successful drivers by season and overall
- Calculate win rates and podium consistency
- Build performance-based driver ranking system
- Analyze long-term consistency using custom metrics

## 🧠 Key Metrics
- Win Rate (%)
- Top 5 Finish Rate
- Average Finishing Position
- Performance Index (custom scoring model)

## 🛠️ Technologies Used
- MySQL
- SQL (CTE, Aggregate Functions, CASE statements, Window functions)

## 📁 Dataset
Formula 1 race results dataset containing:
- race results
- driver names
- seasons
- finishing positions

## 🔥 Example Insights
- Drivers ranked by win rate across seasons
- Most consistent top performers (Top 5 finishes)
- Season-based championship analysis

## 🚀 What I Learned
- Advanced SQL querying techniques
- Data aggregation and transformation
- Building analytical KPIs from raw datasets
- Performance scoring systems
